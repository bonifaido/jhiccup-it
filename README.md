jhiccup-your-os
===============

This small test runs jHiccup against your favorite OS and JVM, and it's also a small demonstration how to use [jHiccup](http://www.azulsystems.com/jHiccup).

## Prerequisites
Your favourite JVM (java) should in be `PATH`.

## Usage

```
./measure.sh
```
